Projeto: Automação de Prestação de Contas
Script desenvolvido para automatizar processos repetitivos e garantir a precisão em rotinas de back-office.

⚙️ Funcionalidades:
Automação: Processamento de dados de prestação de contas, reduzindo trabalho manual.

Tratamento de Erros: Lógica robusta com if/else e tratamento de exceções para garantir que a rotina não pare em caso de dados inconsistentes.

Otimização: Foco em eficiência de processos, alinhado com as necessidades de Service Desk e Projetos.
